h1
A level one heading element, used to define the most important heading
<h1>This is a level one heading example</h1>
h2
The h2 element is used to indicate a heading whose level of importance is exceeded only by h1
<h2>This is a level two heading example</h2>
h3
The h3 element is used to indicate a heading whose level of importance is exceeded by h1 and h2.
<h3>This is a level three heading example</h3>
p
indicates a paragraph of text
<p>This is a paragraph example</p>
ul
an unordered list of items or text
<ul>This is a unordered list example</ul>
ol
an ordered list of items/text
<ol>This is a ordered list of items example</ol>
li
indicates an individual item in a list
<li>This is a individual item in a list example</li>
